Инструкция по установке компонентов для FreezeGen

Шаг 1: Загрузка Python

Перейдите на страницу загрузки Python и выберите версию Python 3.12.4.

Скачать Python 3.12.4

Шаг 2: Запуск Python

Запустите установочный файл Python, который вы скачали.

Шаг 3: Настройки установки

На экране установки выберите опцию "Add Python to PATH" и нажмите "Install Now".

Шаг 4: Завершение установки

Подождите, пока Python установится. После завершения установки нажмите "Close".

Шаг 5: Открытие папки с программой

Откройте папку, где установлена программа FreezeGen.

Шаг 6: Запуск установочных скриптов

В папке FreezeGen найдите файл pip install.bat и дважды щелкните по нему. Подождите, пока необходимые библиотеки будут загружены.

Шаг 7: Запуск FreezeGen

После завершения загрузки запустите start.bat, чтобы начать использование FreezeGen.
